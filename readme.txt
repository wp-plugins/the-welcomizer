This plugin helps you to easily add 'Smart' moves and custom effects to almost any HTML element.

== Description ==

Triggered on the home page, this plugin helps you to easily add 'Smart' moves and custom effects to almost any HTML element that have an ID attribute(e.g. div id="sidebar").

More information can be found at http://www.sebastien-laframboise.com/wordpress/plugins-wordpress/the-welcomizer/

If it's not already done, take a look at my Welcomizer effects on my home page -> http://www.sebastien-laframboise.com

Enjoy!
 
== Features ==

* Activate or deactivate a movement on the fly.
* Delay and duration of a movement are in milliseconds.
* Optional starting position: Top, Left.
* Optional css position: nothing, relative(by default), absolute
* Flexible and easy movement to config (+ or -) Top and Left.
* Optional second move on the same editing panel. (Usefull to make an easy back and forth.)
* Optional custom options textbox, for more custom options.( Google -> jquery animate )
* Optional custom JavaScript textbox, triggered after each move.
* Unlimited number of movements( I don't recommend more than 3 movements with the same time delay)
* Clean uninstallation, means all the data is erased on plugin deactivation.

== Installation ==

1. Upload `/the-welcomizer` directory and files to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Find the plugin under the menu 'Appearance'
